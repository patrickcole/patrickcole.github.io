  • Loot Tracker

    View

    React Application with MongoDB Back-End

    Utility application to keep track of antiques and "loot" when shopping for the best deals. Idea for project came after trying to keep track of prices inside document became complicated. Content is served from back-end MongoDB database in the cloud and rendered on the front-end via ReactJS.

    In addition to listing products, locations and products, the application has the ability to add and edit existing data via a REST API created with NodeJS and Express.

    Maps are embedded into location pages via the `pigeon-maps` package and OpenStreetMap.org tiles to show precise location of where loot items are listed.

  • React-OST

    View

    Single Page Application using React

    Provides the ability to listen to official soundtracks (OST) of video games from YouTube. ReactJS provides the front-end code while a MongoDB database, hosted externally, provides the albums, tracks and Youtube video information. YouTube videos handle the playback and are embedded to application via YouTube embed API.

    Users can save favorites from the listed albums and quickly access albums from the sidebar. Formatting of track information required building a tool to convert text descriptions from YouTube video page into normalized data format.
